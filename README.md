# online-retail-eda
Exploratory data analysis of Online retail store data using R 


## Data source
https://archive.ics.uci.edu/ml/datasets/online+retail#

## Data descriprion
This is a transnational data set which contains all the transactions occurring between 01/12/2010 and 09/12/2011 for a UK-based and registered non-store online retail.


## What I did

This was a blind exploratory data analysis prject, and I mean by blind that I had no access to personnel I could ask about any obscurity in the data.
So, this could be considered useful and what one can do for a company acquiring this online retail store and inheriting their books with minimal information about the data.
* I explored the data in order to understand it first, analyze it second, and provide insightful suggestions at the end.
* Understanding and cleaning the data was the bulk of the work, because there are a lot of inconsistencies in the data. Not only did I present the findings and how I reached them, but I also shared the whole process: trying to understand the data through one variable, meeting a dead end, switching to another approach and using another variable.
* After cleaning the data, and transforming it to a star-like structure, I analyzed the data and used visuals to summarize critical information about sales, discounts, returns, and losses.
* Building on the information gained from the analysis, I provided some actionable suggestions regarding:
  + How to improve the data structure to avoid losing information.
  + How to improve data entry process to avoid accounting mistakes.
  + How to alter discount policy.
  + How to decrease return rate for some products.
